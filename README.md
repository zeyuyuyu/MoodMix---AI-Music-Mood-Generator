# MoodMix

MoodMix is an engaging web app that generates custom music playlists based on your current mood and activities. Simply drag the emoji sliders to express how you're feeling, and our AI will create the perfect playlist for your moment.

Key Features:
- Intuitive emoji mood sliders
- Real-time playlist generation
- Share your mood mixes on social media
- Save favorite mood-playlist combinations

Tech Stack:
- HTML5/CSS3 with animations
- Vanilla JavaScript
- Web Audio API for sound previews
- Local Storage for favorites
